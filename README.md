# present

A terminal presentation tool.

## Installation

```bash
$ pip install present
```

## Usage

```bash
$ present slides.md
```

## Features

### Headers and normal text

```markdown
# present

A terminal presentation tool.
```

### Lists

```markdown
- Basic controls
    - Quit: q
    - Next slide: n, Right arrow
    - Previous slide: b, Left arrow
```

### Code blocks

```python
>>> bin(termios.ICANON)
'0b10'
>>> bin(termios.ECHO)
'0b1000'
```
